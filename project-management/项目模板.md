# Remotion 项目模板库

## 📦 模板概述

项目模板是预配置的视频项目框架，可以快速启动新项目，避免重复配置。

## 🎨 自定义模板创建指南

### 步骤 1：定义模板配置

在 `projects/projects.json` 中添加新模板：

```json
{
  "id": "social-media-short",
  "name": "社交媒体短视频",
  "description": "适合 TikTok、Instagram Reels 等平台的竖版视频",
  "config": {
    "durationInFrames": 180,
    "fps": 30,
    "width": 1080,
    "height": 1920
  }
}
```

### 步骤 2：创建模板组件结构

```
src/projects/TemplateProject/
├── index.tsx           # 主组件
├── config.ts          # 配置文件
├── schema.ts          # Schema 定义
└── scenes/            # 场景组件
    ├── Scene1.tsx
    ├── Scene2.tsx
    └── Scene3.tsx
```

### 步骤 3：实现模板组件

```tsx
// src/projects/TemplateProject/index.tsx
import { AbsoluteFill, Sequence } from "remotion";
import { z } from "zod";
import { Scene1 } from "./scenes/Scene1";
import { Scene2 } from "./scenes/Scene2";
import { Scene3 } from "./scenes/Scene3";

export const templateProjectSchema = z.object({
  title: z.string(),
  subtitle: z.string(),
  showLogo: z.boolean().default(true),
});

export type TemplateProjectProps = z.infer<typeof templateProjectSchema>;

export const TemplateProject = ({ 
  title, 
  subtitle,
  showLogo 
}: TemplateProjectProps) => {
  return (
    <AbsoluteFill>
      <Sequence from={0} durationInFrames={60}>
        <Scene1 title={title} showLogo={showLogo} />
      </Sequence>
      
      <Sequence from={60} durationInFrames={60}>
        <Scene2 subtitle={subtitle} />
      </Sequence>
      
      <Sequence from={120} durationInFrames={60}>
        <Scene3 />
      </Sequence>
    </AbsoluteFill>
  );
};
```

## 🎬 场景模板

### 开场场景模板

```tsx
export const OpeningScene = ({ title, subtitle }: OpeningProps) => {
  const frame = useCurrentFrame();
  const opacity = interpolate(frame, [0, 30], [0, 1]);
  
  return (
    <AbsoluteFill style={{ opacity }}>
      <h1>{title}</h1>
      <p>{subtitle}</p>
    </AbsoluteFill>
  );
};
```

### 结尾场景模板

```tsx
export const ClosingScene = ({ callToAction }: ClosingProps) => {
  return (
    <AbsoluteFill style={{ background: "black" }}>
      <h2>{callToAction}</h2>
    </AbsoluteFill>
  );
};
```

### 过渡场景模板

```tsx
export const TransitionScene = ({ fromColor, toColor }: TransitionProps) => {
  const frame = useCurrentFrame();
  const color = interpolateColors(
    frame,
    [0, 30],
    [fromColor, toColor]
  );
  
  return (
    <AbsoluteFill style={{ background: color }} />
  );
};
```

## 🔧 模板最佳实践

### 1. 参数化设计

使用 Zod Schema 确保类型安全：

```typescript
export const templateSchema = z.object({
  // 必填参数
  title: z.string(),
  duration: z.number(),
  
  // 可选参数
  showLogo: z.boolean().default(true),
  backgroundColor: z.string().default("#ffffff"),
  
  // 复杂参数
  sections: z.array(z.object({
    title: z.string(),
    content: z.string(),
    duration: z.number()
  }))
});
```

### 2. 场景模块化

将复杂视频拆分为可复用的场景组件：

```tsx
export const VideoTemplate = ({ sections }: TemplateProps) => {
  let currentFrame = 0;
  
  return (
    <AbsoluteFill>
      {sections.map((section, index) => {
        const sequence = (
          <Scene 
            key={index}
            from={currentFrame}
            durationInFrames={section.duration}
            {...section}
          />
        );
        currentFrame += section.duration;
        return sequence;
      })}
    </AbsoluteFill>
  );
};
```

### 3. 主题系统

```typescript
const themes = {
  light: {
    background: "#ffffff",
    text: "#000000",
    accent: "#007bff"
  },
  dark: {
    background: "#1a1a1a",
    text: "#ffffff",
    accent: "#4dabf7"
  }
};

export const ThemedTemplate = ({ 
  theme = "light",
  children 
}: ThemedProps) => {
  const colors = themes[theme];
  return (
    <AbsoluteFill style={{ 
      background: colors.background,
      color: colors.text 
    }}>
      {children}
    </AbsoluteFill>
  );
};
```
