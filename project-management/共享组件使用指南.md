# 🔧 共享组件使用指南

## 📖 概述

`shared-components` 包包含了可在多个 Remotion 项目中复用的公共组件，避免重复代码，提高开发效率。

---

## 📦 当前可用组件

### 1. TechGridBackground（科技网格背景）

一个高科技感的网格背景组件，专为赛博朋克风格设计。

#### 组件特性
- ✨ 3D 透视网格效果
- 🌟 持续移动动画
- 💫 可调节发光效果
- 🎯 粒子系统
- 🎨 完全可自定义
- 📱 支持任意分辨率

---

## 🚀 快速使用

### 在项目中导入共享组件

```typescript
// 1. 在项目的 package.json 中添加依赖
{
  "dependencies": {
    "shared-components": "workspace:*"
  }
}

// 2. 在组件中导入使用
import { TechGridBackground, techGridBackgroundSchema } from 'shared-components';

// 3. 使用组件
<TechGridBackground
  gridColor="#00ffff"
  glowIntensity={0.8}
  speed={1}
  showParticles={true}
  showTitle={true}
  title="MY TITLE"
  subtitle="Subtitle"
/>
```

---

## 🎯 TechGridBackground 参数详解

### 基础配置参数

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `gridColor` | string | "#00ffff" | 网格颜色（十六进制） |
| `glowIntensity` | number | 0.8 | 发光强度（0-1） |
| `speed` | number | 1 | 移动速度（0.1-3） |
| `gridSize` | number | 60 | 网格大小（20-100像素） |
| `perspective` | number | 800 | 透视距离（400-1200） |

### 显示选项

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `showParticles` | boolean | true | 显示粒子效果 |
| `showScanline` | boolean | true | 显示扫描线效果 |
| `showTitle` | boolean | false | 显示标题内容 |

### 内容参数

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `title` | string | undefined | 主标题 |
| `subtitle` | string | undefined | 副标题 |

### 布局参数

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `width` | number | 1440 | 容器宽度 |
| `height` | number | 2560 | 容器高度 |

## 💡 使用示例

### 示例 1：简单背景

```tsx
import { AbsoluteFill } from "remotion";
import { TechGridBackground } from "shared-components";

export const MyVideo = () => {
  return (
    <AbsoluteFill>
      <TechGridBackground
        gridColor="#00ffff"
        glowIntensity={0.8}
        speed={1}
      />
      {/* 你的内容 */}
    </AbsoluteFill>
  );
};
```

### 示例 2：带标题的背景

```tsx
export const MyTitledVideo = () => {
  return (
    <AbsoluteFill>
      <TechGridBackground
        gridColor="#ff1493"
        glowIntensity={1.0}
        speed={0.8}
        showTitle={true}
        title="CYBERPUNK"
        subtitle="NEON DREAMS"
      />
    </AbsoluteFill>
  );
};
```

### 示例 3：自定义配置

```tsx
export const CustomTechVideo = () => {
  return (
    <AbsoluteFill>
      <TechGridBackground
        gridColor="#9d4edd"
        glowIntensity={0.6}
        speed={1.5}
        gridSize={40}
        perspective={1000}
        showParticles={true}
        showScanline={true}
        showTitle={false}
        width={1920}
        height={1080}
      />
    </AbsoluteFill>
  );
};
```

### 示例 4：在 Composition 中使用

```tsx
import { Composition } from "remotion";
import { TechGridBackground, techGridBackgroundSchema } from "shared-components";

export const RemotionRoot = () => {
  return (
    <Composition
      id="MyTechVideo"
      component={() => (
        <TechGridBackground
          gridColor="#00ffff"
          glowIntensity={0.8}
          speed={1}
          showTitle={true}
          title="TECH NOLOGY"
          subtitle="Future is Now"
        />
      )}
      durationInFrames={300}
      fps={30}
      width={1440}
      height={2560}
    />
  );
};
```

## 📊 性能优化建议

### 渲染性能
- **降低网格密度**：增加 `gridSize` 值（推荐 80-100）
- **减少粒子**：设置 `showParticles={false}`
- **降低发光强度**：设置 `glowIntensity={0.5}`

### 不同分辨率优化

```typescript
// 高分辨率（2K+）
<TechGridBackground gridSize={80} glowIntensity={0.8} />

// 中等分辨率（1080p）
<TechGridBackground gridSize={60} glowIntensity={0.6} />

// 低分辨率（720p）
<TechGridBackground gridSize={40} glowIntensity={0.5} />
```

## 🐛 故障排除

### Q: 组件导入失败？
**A**: 确保在 package.json 中添加了 `"shared-components": "workspace:*"` 并运行 `pnpm install`。

### Q: 样式没有生效？
**A**: 确保导入了组件的 CSS 文件或样式已正确包含。

### Q: 性能太慢？
**A**: 尝试增加 `gridSize` 值、关闭粒子效果、降低发光强度。

### Q: 如何修改默认参数？
**A**: 直接在组件调用时传入参数覆盖默认值。
