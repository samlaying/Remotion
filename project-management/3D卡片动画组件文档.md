# 🎴 3D 卡片旋转动画组件

## 📋 组件概述

**ThreeDCard** 是一个功能强大的3D卡片旋转动画组件，支持透明背景导出，专为创建产品展示、图标动画等设计。

---

## 🎨 组件特性

### 核心功能
- 🎯 **3D 旋转动画**：支持X轴、Y轴旋转
- ✨ **透明背景**：可导出带透明通道的视频
- 🎪 **浮动效果**：自然的上下浮动动画
- 🔄 **倾斜效果**：动态的卡片倾斜
- 💫 **光影效果**：阴影、反射、发光效果
- 🎨 **完全可定制**：30+ 可配置参数

### 技术规格
- **分辨率**：1080x1080 (正方形)
- **帧率**：30 FPS
- **时长**：300 帧 (10 秒)
- **输出格式**：PNG (支持透明通道)

---

## 🚀 快速使用

### 启动项目

```bash
# 启动3D卡片项目
pnpm dev:card

# 访问地址
http://localhost:3003
```

### 预设主题

项目包含 6 个预设主题：

1. **BasicRotationTransparent** - 基础旋转（透明背景）
2. **DualAxisDarkBg** - 双轴旋转（黑色背景）
3. **FastRotationGradient** - 快速旋转（渐变背景）
4. **GentleSwayClean** - 轻微摆动（透明背景）
5. **StrongEffectsDark** - 强烈光效（深色背景）
6. **SlowShowcaseTransparent** - 慢速展示（透明背景）

## 🔧 组件参数详解

### 卡片内容
```typescript
{
  imageSrc: string;           // 图片路径（必填）
  title?: string;             // 主标题（可选）
  subtitle?: string;          // 副标题（可选）
}
```

### 3D 动画配置
```typescript
{
  rotateY: boolean;           // Y轴旋转（默认：true）
  rotateX: boolean;           // X轴旋转（默认：false）
  floatAnimation: boolean;    // 浮动动画（默认：true）
  tiltEffect: boolean;        // 倾斜效果（默认：true）
}
```

### 旋转参数
```typescript
{
  rotationSpeed: number;      // 旋转速度（默认：1，范围：0.1-3）
  maxRotationAngle: number;   // 最大旋转角度（默认：360）
  floatAmplitude: number;     // 浮动幅度（默认：20像素）
  floatSpeed: number;         // 浮动速度（默认：1）
}
```

### 卡片样式
```typescript
{
  cardWidth: number;          // 卡片宽度（默认：400像素）
  cardHeight: number;         // 卡片高度（默认：400像素）
  cardBorderRadius: number;   // 圆角半径（默认：20像素）
  cardScale: number;          // 缩放比例（默认：1）
  cardOpacity: number;        // 透明度（默认：1）
}
```

### 光影效果
```typescript
{
  shadowIntensity: number;    // 阴影强度（默认：0.3，范围：0-1）
  reflectionIntensity: number;// 反射强度（默认：0.1，范围：0-1）
  glowIntensity: number;      // 发光强度（默认：0，范围：0-1）
  glowColor: string;          // 发光颜色（默认："#00ff00"）
}
```

### 背景设置 ⭐
```typescript
{
  transparentBackground: boolean;  // 透明背景（默认：false）⭐
  backgroundColor: string;         // 背景颜色（默认："#000000"）
  backgroundGradient: boolean;     // 渐变背景（默认：false）
  gradientColors: string[];        // 渐变颜色（默认：["#000000", "#1a1a1a"]）
}
```

### 视角设置
```typescript
{
  perspective: number;        // 透视距离（默认：1000像素）
  cameraDistance: number;     // 相机距离（默认：500像素）
}
```

### 时序控制
```typescript
{
  entranceAnimation: boolean; // 入场动画（默认：true）
  entranceDuration: number;   // 入场时长（默认：60帧）
  loopAnimation: boolean;     // 循环动画（默认：true）
}
```

## 🎬 渲染透明背景视频

### 渲染单个透明背景视频

```bash

cd packages/project-3d-card
```

### 渲染所有变体

```bash
# 渲染所有6个预设主题
pnpm render:all
```

### 自定义渲染参数

```bash
# 渲染特定时长
npx remotion render BasicRotationTransparent output/output.mp4 --frames=600

npx remotion render BasicRotationTransparent output/output.mp4 --width=1920 --height=1920

# 渲染不同帧率
npx remotion render BasicRotationTransparent output/output.mp4 --fps=60
```

## ⭐ 透明背景详解

### 什么是透明背景？

透明背景是指视频背景完全透明，可以叠加到其他内容上，非常适合：
- 🎬 视频编辑合成
- 🌐 网页动画叠加
- 📱 应用内动画
- 🎮 游戏UI元素

### 如何启用透明背景？

在组件参数中设置：
```typescript
{
  transparentBackground: true  // ⭐ 关键参数
}
```

### 透明背景的注意事项

1. **输出格式**：使用 PNG 或 ProRes 4444 格式
2. **文件大小**：透明视频文件较大
3. **兼容性**：确保目标平台支持透明视频
4. **性能**：渲染透明视频需要更多资源

### 透明背景最佳实践

```typescript
// ✅ 推荐：透明背景 + 简洁效果
{
  transparentBackground: true,
  shadowIntensity: 0.2,      // 减少阴影
  glowIntensity: 0,          // 移除发光
  cardScale: 0.8,            // 适当缩放
}

// ❌ 避免：透明背景 + 强烈效果
{
  transparentBackground: true,
  shadowIntensity: 0.8,      // 过强阴影
  glowIntensity: 1.0,        // 过强发光
  // 这些效果在透明背景下可能不自然
}
```

## 🔧 故障排除

### Q: 透明背景不透明？
**A**: 检查以下几点：
1. 确保 `transparentBackground: true`
2. 检查渲染格式是否支持透明通道
3. 确认阴影和发光强度不会影响透明度

### Q: 动画太慢？
**A**: 调整以下参数：
```typescript
{
  rotationSpeed: 2,        // 增加旋转速度
  floatSpeed: 1.5,         // 增加浮动速度
  entranceDuration: 30,    // 减少入场时长
}
```

### Q: 卡片太小？
**A**: 调整缩放参数：
```typescript
{
  cardScale: 1.2,          // 增加缩放
  perspective: 800,        // 减少透视距离
}
```

### Q: 效果不够明显？
**A**: 增强光影效果：
```typescript
{
  shadowIntensity: 0.6,    // 增加阴影
  glowIntensity: 0.8,      // 增加发光
  reflectionIntensity: 0.2 // 增加反射
}
```

## 📊 性能优化

### 渲染性能
- **减少阴影**：降低 `shadowIntensity`
- **移除发光**：设置 `glowIntensity: 0`
- **简化动画**：减少同时运行的动画效果

### 文件大小优化
- **缩短时长**：减少 `durationInFrames`
- **降低分辨率**：使用较小的宽高
- **压缩设置**：后期压缩视频文件
