# 🌐 科技网格背景组件

## 📋 组件概述

**TechGrid** 是一个高科技感的网格背景动画组件，专为创建赛博朋克风格、科技主题的视频设计。

---

## 🎨 组件特性

### 视觉效果
- ✨ **3D 透视网格**：具有深度感的透视网格效果
- 🌟 **持续移动动画**：网格线持续向远处移动
- 💫 **发光效果**：网格线带有科技感发光
- 🎯 **粒子系统**：浮动粒子增强科技氛围
- 🌈 **多种颜色主题**：支持自定义颜色配置
- 📱 **竖屏优化**：专为 6:9 竖屏设计

### 技术规格
- **分辨率**：1440x2560 (2K 竖屏)
- **比例**：6:9
- **帧率**：30 FPS
- **时长**：300 帧 (10 秒)

## 🔧 组件参数

### Schema 定义

```typescript
export const techGridSchema = z.object({
  title: z.string(),                    // 主标题
  subtitle: z.string().optional(),      // 副标题（可选）
  gridColor: z.string().default("#00ffff"),  // 网格颜色
  glowIntensity: z.number().default(0.8),    // 发光强度 (0-1)
  speed: z.number().default(1),               // 移动速度 (0.5-2)
});
```

### 参数说明

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `title` | string | 必填 | 主标题文字 |
| `subtitle` | string | 可选 | 副标题文字 |
| `gridColor` | string | "#00ffff" | 网格颜色（十六进制） |
| `glowIntensity` | number | 0.8 | 发光强度（0-1） |
| `speed` | number | 1 | 移动速度（0.5-2） |

## 🎯 使用示例

### 创建自定义主题

```tsx
import { Composition } from "remotion";
import { TechGrid, techGridSchema } from "./index";

<Composition
  id="MyCustomTechGrid"
  component={TechGrid}
  durationInFrames={300}
  fps={30}
  width={1440}
  height={2560}
  schema={techGridSchema}
  defaultProps={{

    subtitle: "Custom Subtitle",
    gridColor: "#ff1493",    // 深粉色
    glowIntensity: 1.0,      // 最大发光

/>
```

## 🎨 动画效果详解

### 1. 网格移动动画
- **水平线**：从下往上持续移动
- **垂直线**：从左往右缓慢移动
- **透视效果**：3D 透视增强深度感

### 2. 发光脉冲
- **呼吸效果**：发光强度在 0.6-1.0 之间波动
- **频率**：每 3 秒一个周期
- **强度**：可通过 `glowIntensity` 参数调整

### 3. 粒子系统
- **数量**：50 个浮动粒子
- **运动**：从下往上浮动，逐渐消失
- **分布**：黄金角度均匀分布
- **发光**：每个粒子带有发光效果

### 4. 扫描线效果
- **位置**：从上到下移动
- **周期**：3 秒
- **透明度**：淡入淡出效果

## 🐛 常见问题

### Q: 如何修改分辨率？
**A**: 在 `Root.tsx` 中修改 Composition 的 `width` 和 `height` 参数。

### Q: 网格移动速度如何调整？
**A**: 修改 `defaultProps` 中的 `speed` 参数（0.5-2 之间）。

### Q: 如何去除粒子效果？
**A**: 在 `index.tsx` 中注释掉粒子部分的代码。

### Q: 发光效果太强如何调整？
**A**: 降低 `glowIntensity` 参数值（建议 0.3-0.8）。
