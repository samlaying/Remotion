# Remotion 项目管理系统使用指南

## 📖 系统概述

这是一个基于 Remotion 的视频项目管理系统，可以创建、管理和切换多个不同的视频项目。

### 核心特性

- 🎯 **多项目管理**：在一个 Remotion 项目中管理多个视频项目
- 📦 **模板系统**：提供预设模板，快速创建新项目
- 🔧 **配置化管理**：所有项目配置集中管理
- 🚀 **动态加载**：根据项目状态动态加载视频

## 🚀 快速开始

### 1. 查看现有项目

```bash
npm run dev
```

启动后在浏览器中打开 http://localhost:3000，你会看到所有活跃项目的列表。

### 2. 创建新项目

#### 方法一：使用模板

在 `projects/projects.json` 中添加新项目：

```json
{
  "id": "my-new-project",
  "name": "我的新项目",
  "description": "项目描述",
  "category": "tutorial",
  "status": "active",
  "config": {
    "durationInFrames": 300,
    "fps": 30,
    "width": 1920,
    "height": 1080
  },
  "schema": {},
  "defaultProps": {}
}
```

#### 方法二：使用代码

```typescript
import { createProjectFromTemplate } from "./lib/projectManager";

const newProject = createProjectFromTemplate("basic-video", {
  name: "我的新项目",
  description: "这是一个新项目",
  category: "demo"
});
```

### 3. 实现项目组件

在 `src/projects/MyNewProject/index.tsx` 中创建项目组件：

```tsx
import { AbsoluteFill } from "remotion";
import { z } from "zod";

export const myProjectSchema = z.object({
  title: z.string(),
  subtitle: z.string(),
});

export type MyProjectProps = z.infer<typeof myProjectSchema>;

export const MyNewProject = ({ title, subtitle }: MyProjectProps) => {
  return (
    <AbsoluteFill style={{ background: "white" }}>
      <h1>{title}</h1>
      <p>{subtitle}</p>
    </AbsoluteFill>
  );
};
```

### 4. 注册项目

在 `src/Root.tsx` 中导入并注册你的项目：

```tsx
import { MyNewProject, myProjectSchema } from "./projects/MyNewProject";

<Composition
  id="MyNewProject"
  component={MyNewProject}
  durationInFrames={300}
  fps={30}
  width={1920}
  height={1080}
  schema={myProjectSchema}
  defaultProps={{
    title: "我的项目",
    subtitle: "项目描述"
  }}
/>
```

## 📋 项目状态管理

### 项目状态类型

- **active**: 活跃项目，会在 Studio 中显示
- **draft**: 草稿项目，暂时隐藏
- **archived**: 归档项目，不再使用

### 切换项目状态

```typescript
import { updateProject } from "./lib/projectManager";

// 将项目设为活跃状态
updateProject("project-id", { status: "active" });

// 将项目设为草稿状态
updateProject("project-id", { status: "draft" });
```

## 📝 最佳实践

### 1. 项目命名规范

- 使用小写字母和连字符：`my-project-name`
- 避免使用特殊字符和空格
- 使用描述性的名称

### 2. 项目分类

推荐的项目分类：
- `tutorial`: 教程类视频
- `demo`: 演示类视频
- `promo`: 促销类视频
- `social`: 社交媒体视频
- `internal`: 内部使用视频

### 3. 项目组织

每个项目应该包含：
- 清晰的描述
- 合适的分类
- 正确的状态
- 完整的配置
- 类型安全的 Props

### 4. 版本控制

- 定期更新 `updatedAt` 字段
- 重要变更时更新版本号
- 保持项目配置同步

## 🎯 渲染项目

### 渲染单个项目

```bash
# 渲染 RemotionGuide 项目
remotion render RemotionGuide output/remotion-guide.mp4

# 渲染自定义项目
remotion render MyProject output/my-project.mp4
```

### 批量渲染

创建脚本批量渲染所有活跃项目：

```bash
#!/bin/bash
# render-all.sh

projects=("RemotionGuide" "MyProject" "AnotherProject")

for project in "${projects[@]}"
do
  echo "Rendering $project..."
  remotion render $project output/$project.mp4
done
```

## 🐛 常见问题

### Q: 项目没有在 Studio 中显示？
A: 检查项目的 `status` 是否为 `active`，以及是否正确在 `Root.tsx` 中注册。

### Q: 修改项目配置后没有生效？
A: 重启开发服务器，确保配置文件正确保存。

### Q: 如何备份项目数据？
A: 定期备份 `projects/projects.json` 文件和 `src/projects/` 目录。
