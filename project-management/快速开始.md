# 🚀 快速开始指南

## 1. 启动开发服务器

```bash
npm run dev
```

在浏览器中打开 http://localhost:3000

---

## 2. 查看现有项目

```bash
# 使用交互式项目管理器
npm run project

# 或直接列出项目
npm run project:list
```

---

## 3. 创建新项目

### 方法一：使用交互式命令

```bash
npm run project:create
```

按照提示输入项目信息：
- 项目名称
- 项目描述  
- 项目分类
- 是否使用模板
- 配置参数

### 方法二：手动创建

1. 在 `projects/projects.json` 中添加项目配置：

```json
{
  "id": "my-awesome-project",
  "name": "我的精彩项目",
  "description": "这是一个很棒的视频项目",
  "category": "demo",
  "status": "active",
  "config": {
    "durationInFrames": 300,
    "fps": 30,
    "width": 1920,
    "height": 1080
  },
  "schema": {},
  "defaultProps": {}
}
```

2. 在 `src/projects/MyAwesomeProject/index.tsx` 中创建组件：

```tsx
import { AbsoluteFill } from "remotion";
import { z } from "zod";

export const myAwesomeProjectSchema = z.object({
  title: z.string(),
});

export type MyAwesomeProjectProps = z.infer<typeof myAwesomeProjectSchema>;

export const MyAwesomeProject = ({ title }: MyAwesomeProjectProps) => {
  return (
    <AbsoluteFill style={{ background: "white" }}>
      <h1>{title}</h1>
    </AbsoluteFill>
  );
};
```

3. 在 `src/Root.tsx` 中注册项目：

```tsx
import { MyAwesomeProject, myAwesomeProjectSchema } from "./projects/MyAwesomeProject";

<Composition
  id="MyAwesomeProject"
  component={MyAwesomeProject}
  durationInFrames={300}
  fps={30}
  width={1920}
  height={1080}
  schema={myAwesomeProjectSchema}
  defaultProps={{ title: "我的精彩项目" }}
/>
```

## 4. 渲染项目

### 渲染单个项目

```bash
# 使用 npm script
npm run render

# 或使用 remotion CLI
npx remotion render MyProject output/my-project.mp4
```

### 批量渲染所有项目

```bash
npm run render:all
```

这会将所有活跃项目渲染到 `output/` 目录。

## 5. 项目管理

### 交互式项目管理

```bash
npm run project
```

这将打开一个交互式菜单：
1. 列出所有项目
2. 创建新项目
3. 删除项目
4. 退出

### 命令行操作

```bash
# 列出项目
npm run project:list

# 创建项目
npm run project:create

# 删除项目
npm run project:delete
```

## 6. 开发工作流

### 典型的一天

1. **启动开发服务器**
   ```bash
   npm run dev
   ```

2. **在浏览器中编辑**
   - 打开 http://localhost:3000
   - 选择项目进行编辑
   - 实时预览效果

3. **测试参数**
   - 在右侧面板修改参数
   - 查看实时效果
   - 确定最佳参数

4. **类型检查**
   ```bash
   npm run typecheck
   ```

5. **渲染最终视频**
   ```bash
   npm run render
   ```

## 8. 故障排除

### 项目没有在 Studio 中显示

1. 检查项目状态是否为 `"active"`
2. 确认在 `Root.tsx` 中已注册
3. 重启开发服务器

### 修改配置后没有生效

1. 重启开发服务器
2. 清除浏览器缓存
3. 检查 JSON 文件格式

### 渲染失败

1. 检查项目配置参数
2. 验证 Schema 定义
3. 确认组件正确导出
